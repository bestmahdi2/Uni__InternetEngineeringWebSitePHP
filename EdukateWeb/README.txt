Requirements:

XAMPP v3.3.0
PHP/MySQLnd 8.2.4
Apache 2.4.56
MariaDB 10.4.28

--------------

How To Run:

1. Open XAMPP Control Panel
2. Start Apache and MySQL
3. Create "edukate" database
4. Import "database/Database.sql" file into "edukate" database, it will create all the tables
5. Place "EdukateWeb" into "C:\xampp\htdocs\"
6. Open this link in your browser "Localhost/EdukateWeb/index.php"
